# 术语表

## AGENTS.md

用于提供系统级或项目级行为指令的 Markdown 文件（Codex/OpenCode 等工具可读取）。

## CLAUDE.md

Claude Code 的记忆/指令文件，用于约束项目上下文和工作方式。

## MCP

Model Context Protocol。用于把模型与外部工具系统连接。

## Skill

可复用的任务能力单元，通常包含结构化说明和执行流程。

## Sandbox

隔离执行环境，用于限制文件系统、网络和命令权限范围。

## Claim

页面中一个可验证的事实性陈述，使用 `claim_id` 统一标识。

## Citation

对 claim 的证据引用，通常在页面中呈现为上标 `[n]`。

## Verification Status

claim 的验证状态：`verified`、`partially_verified`、`conflicted`、`unverified`。
