2. settings.json → config.toml

3. 重组 Session 目录（按日期）

4. 迁移 Skills → 自定义脚本

1. 拆分 CLAUDE.md → .opencode/agents/

3. MCPs → .opencode/tools/

4. 配置多 LLM 提供商
